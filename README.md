Nombre proyecto: Ecommerce Market V1.0.0

Descripcion proyecto: Ecommerce dedicado a la venta de productos de supermercado.

Componentes creados:
- Banner
- Navbar
- SearchInput
- Card
- ImageStatus
- Button
- Footer
- ProductList
- ProductDetail

Instrucciones para ejecutar el proyecto:
1- Descargue el proyecto desde el repositorio de Github
2- Abra la Terminal en MACOS o Linux, CMD en Windows.
3- Dirijase al directorio donde descargo el proyecto.
4- Ejecute en la Terminal o CMD el siguiente comando: npm install
5- Ejecute en la Terminal o CMD el siguiente comando: npm run dev
6- Por ultimo copie la direccion web que le entrega la consola (http://localhost:5173/) en el navegador que utilice.
7- El proyecto se ejecutara en su navegador.

Tecnologias usadas: ReactJS, Vite, CSS, HTML.
